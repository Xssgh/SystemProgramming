0314 필기 내용

/ 	: root directory
~ 	: home directory
. 	: self
.. 	: parent

경로의 종류
1. 절대 경로 : cd /home/jang/software
2. 상대 경로 : cd ./software 또는 cd software

새로운 디렉토리 만들기 : mkdir
새로운 텍스트 파일 만들기 : cat > (텍스트 파일 이름), 원하는 내용 작성 후 "ctrl + d"
